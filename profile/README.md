# FirstLayer

**Der smarte Chat-Layer für Streamer — direkt über deinem Spiel.**

FirstLayer legt deinen Twitch-Chat als Overlay über dein Spiel, damit du nichts mehr verpasst, ohne ständig den Blick zu wechseln. Aber FirstLayer ist mehr als nur ein Overlay: der Chat wird intelligent aufbereitet — doppelte Messages werden zusammengefasst, wichtige Nachrichten hervorgehoben, Events landen in einer eigenen Activity-Liste. Und mit **Pulse**, unserer AI-Integration, bekommst du Live-Insights zu Stimmung, Topics und deiner Community.

Weniger Chaos. Mehr Überblick. Entspannter streamen.

## ✨ Features

- 🎮 **In-Game Overlay** — Chat direkt über dem Spiel, oder klassisch auf zweitem Monitor / in OBS
- 🧠 **Smart Chat** — Duplicate-Merging ("GG ×20"), Context Mode, First-Message- und Mention-Highlights
- 📋 **Activity Feed** — Raids, Follows & Subs getrennt vom Chat
- 🤖 **Pulse AI** — erkennt Topics, Sentiment und schlägt vor, worauf du reagieren kannst
- 👥 **Viewer Profiles** — Mini-Profile deiner Zuschauer mit Aktivität und Vibes
- 📊 **Stream Insights** — AI-Dashboard, das deine Streams auswertet und dir zeigt, was gut ankam und wo du nachjustieren kannst
- 🎨 **Themes, Alerts & Widgets** — volle Anpassbarkeit
